# DailyTracker Mobile

This is an Expo-based mobile app scaffold for DailyTracker, built to run on both Android and iOS from the same codebase.

## Getting started

1. Install dependencies:

```bash
cd mobile-app
npm install
```

2. Start the app:

```bash
npm start
```

3. Launch on a simulator or physical device from the Expo dashboard.

## Notes

- The app currently includes home, profile, journal, and archive screens.
- This scaffold is designed so you can later share logic or UI with the existing web frontend.
